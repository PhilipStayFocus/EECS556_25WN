How to change the size of kernel for padding training?
1.Cd deraining\lpnet\basicsr\models\archs\LPNet_pad_arch.py , 
2.Change the size in freup_pad class 

How the `LPNet_pad_arch.py ` connected to `train.py`?

1.In `train.py`, there is a `create_model()` function, which is imported from `basicsr.models` 
2. In `create_model()`,  scan the para `model_type` in `*.yml`.
3.Once select the `lpnet_model`, lpnet_model will define the network first.
4.cd deraining\lpnet\build\libwin-amd64-cpython-38\basicsr\models\archs\__init__.py, define_network() load the network structure

In conclusion, train.py connects to lpnet_model.py, and then lpnet_model connects to the corresponding _arch.py.
